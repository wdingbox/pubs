Book GCF Analysis


To get the meaning of TBI book, we apply GCF process. We first got a paticular set (1) from TBI, the most common object occured 7 times in set (1). We took that GCF out and got set (2). In set (2), the most common object occured 4 times. We took that out and got set (3).  Repeating such process until end.  We could get a Frequency Table.

8. Conclusions

The statistic analysis method the paper provided here looks easy and simple. But our study showed that these methods are powerful and played important roles in our TBI study.  For example, through GCF method, more than 610,406 scattered unknown characters collected from Anyan site, that is close to the number of NIV OT (NIVStatistic), can be reduced to become about one thousand root characters.
