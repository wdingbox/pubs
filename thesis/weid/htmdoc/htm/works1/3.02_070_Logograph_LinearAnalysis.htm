  2. Polymorphism
    To use multiple different objects to represent the same meaning, or to use one same image of object to represent different meanins in consistency.

    Single line variation can only populate the limited number of five new characters. To produce more, encapsulation in OOA method has to be applied in order to combine multiple lines to generate new characters. This led to the classification of Self-Clone, Mixed Aggregation (MA).     1) Self-Cloned Aggregation (SCA)
    To form a new character through increase the number of object by duplication of itself. Such children is called SCC.

    2) Mixed Aggregation (MA)
    To form a new character through increase the number of mixed different objects. Such children is called HCC.
