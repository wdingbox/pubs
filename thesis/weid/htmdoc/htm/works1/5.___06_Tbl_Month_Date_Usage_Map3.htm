Table-4
Symbols Specification for Frequency Statistic Table in OBI
Names Descriptions for the symbols in statistic tables.
Tm1 Total number of inscriptions that contains both month-count and day-count. For example, H22847,
戊午卜，冎，貞󾍎󰫪大戊戠亡𡆥。才十四月
Tm0 Total number of inscriptions that only contain month-mount, but without days-count. For example, H21897,
缶。十四月。
Tm Tm = Tm1 + Tm0,
The total number of inscriptions having specific month-count with or without day-count.
Tn1 Total number of inscriptions for the the day with listed-months.
Tn0 Total number of inscriptions for the the day without  listed-months.
Tn Tn = Tn1 + Tn0
 - dash means the day-count was not found for specific month.
 \ backslash: Left site value of the backslash denotes the sum of row. Right site value denotes the sum of column.
